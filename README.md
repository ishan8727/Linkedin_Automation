# LinkedIn Automation Platform

Monorepo for LinkedIn automation platform.

## Structure

- **apps/frontend** - Next.js web application
- **apps/backend** - Node.js API server
- **apps/agent** - Local Playwright automation daemon
- **packages/shared-types** - Shared TypeScript types and interfaces
- **packages/shared-utils** - Common utility functions
- **packages/shared-config** - Shared configuration files
- **antigravity** - Antigravity AI integration
- **docs** - Documentation

## Setup

```bash
# Install dependencies
pnpm install

# Run all apps in development
pnpm dev

# Build all packages
pnpm build
```
